About HTML.

The history of HTML.

HTML was invented by Tim Berners-Lee in 1990 at the CERN research institute.

1991:

Tim Berners-Lee publicly released the first document describing HTML.

1993:

HTML 1.0 was proposed, adding support for embedded images and text formatting.

1995:

HTML 2.0 introduced interactive web forms.

1997:

HTML 3.2 enabled tables and early JavaScript integration.

1999:

HTML 4.01 standardized separation between HTML and CSS.

2000:

XHTML 1.0 was introduced as a stricter XML-based version.

2004:

Developers formed WHATWG to improve HTML.

2014:

HTML5 became the official standard.

What is HTML?

The full form of HTML

HTML stands for HyperText Markup Language.

Hypertext: Text with clickable links.

Markup Language: Tags that structure webpage content.

Why is it used?

HTML is used to structure webpages.

Properties of HTML

  • Not a programming language.
  • Responsible for webpage structure.
  • Works with CSS and JavaScript.
  • Uses tags and elements.
